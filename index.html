Journex

A fully responsive template

Journex is a fully responsive under construction/coming soon template with a modern design suitable for all creative fields. The template is featuring a powerful fullscreen background video and imagery making it a perfect choice for photographers, artists and designers who want to showcase their work while building a website.

Journex makes it easy for you to create a website. It is built on top of the hugely popular Bootstrap framework thus offering clean, semantic HTML and well-structured CSS. The code is excellently formatted and commented, so when it comes to customizing it can be carried out with ease.
